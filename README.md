# Study Docs MCP Server

A Model Context Protocol (MCP) server that allows an AI assistant
to securely access local study documents to answer course-related questions.

## Features
- List available documents
- Read text-based study files
- Keyword search within documents
- Protected folder scope

## Requirements
- Python 3.10+
- fastmcp (`pip install mcp`)
- A folder named `classDocs` with sample documents
